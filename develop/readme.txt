A Custom WordPress Theme for The TinTin Shop, meticulously crafted by Greenwich Design.

== Description ==
TinTin is a custom WordPress theme designed specifically for The TinTin Shop. Emphasizing a blend of elegance and functionality, the theme showcases products with an intuitive user interface. It's a perfect solution for e-commerce sites seeking a unique digital presence.

== Installation ==
1. In your WordPress admin panel, go to Appearance -> Themes and click the 'Add New' button.
2. Click 'Upload Theme' and Choose File, then select the theme's .zip file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==
Q: What version of WordPress do I need?
A: This theme requires at least WordPress 5.2 and is compatible up to WordPress 5.4.

Q: Is this theme compatible with WooCommerce?
A: Yes, TinTin is designed to work seamlessly with WooCommerce.
